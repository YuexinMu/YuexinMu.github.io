LIO-DPC is a LiDAR-inertial odometry framework designed to combine the real-time performance of filtering methods with the accuracy benefits of graph optimization.

The method introduces a dynamic pose chain that allows filtering and local graph optimization to run in parallel. It also uses loop sparsification to select representative high-quality loop closures, reducing computational cost while improving mapping accuracy.

Extensive experiments demonstrate that LIO-DPC achieves real-time performance and improves accuracy compared with state-of-the-art LiDAR-inertial odometry methods.
